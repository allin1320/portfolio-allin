# Quickstart: Add Navbar

**Feature**: Add Navbar  
**Date**: 2025-11-20

## Prerequisites

- Existing Astro + TailwindCSS portfolio project
- Node.js installed

## Steps

1. Create `src/components/Navbar.astro`
2. Update `src/layouts/BaseLayout.astro` to include navbar
3. Add section IDs to existing components if missing
4. Test navigation functionality

## Commands

```bash
npm run dev  # Start development server
npm run build  # Build for production
```
